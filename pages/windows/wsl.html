WLS подсистема Window для Linux

Для хранения дистрибутивов Linux (или файлов Linux - как сказано в документации по WSL ссылка на докумендацию) используется виртуальный жесткий диск (VHD). В WSL виртуальный жесткий диск существует в виде VHDX-файла на жестком диске Windows.
